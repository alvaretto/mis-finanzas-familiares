🏠 Asistente de Configuración

Configura tu App de Finanzas Familiares en 4 pasos fáciles

Progreso Paso 1 de 4
📋

Paso 1: Preparación

Vamos a preparar los archivos necesarios

✅ ¿Ya descargaste el proyecto?

Si no lo has hecho, descárgalo desde:

📥 Descargar Proyecto

📝 Crear archivo de configuración

  1. Busca el archivo config.example.js
  2. Cópialo y pégalo en la misma carpeta
  3. Renombra la copia a config.js
🤖

Paso 2: Configurar IA

Configura Google Gemini para consejos inteligentes

🔑 Obtener API Key

  1. Ve a Google AI Studio
  2. Inicia sesión con tu cuenta Google
  3. Crea una nueva API Key
  4. Copia la API Key generada
🚀 Ir a Google AI Studio

📝 Configurar en tu archivo

En tu archivo config.js, busca:

const GEMINI_API_KEY = "TU_API_KEY_DE_GEMINI_AQUI";

Y reemplaza TU_API_KEY_DE_GEMINI_AQUI con tu API Key real.

🔥

Paso 3: Configurar Firebase

Configura tu base de datos segura

🏗️ Crear proyecto Firebase

  1. Ve a Firebase Console
  2. Crea un nuevo proyecto
  3. Ponle un nombre (ej: "mis-finanzas-familiares")
  4. Desactiva Google Analytics
🚀 Ir a Firebase Console

🔐 Authentication

  1. Clic en "Authentication"
  2. Clic en "Comenzar"
  3. Pestaña "Sign-in method"
  4. Habilitar "Email/password"

💾 Firestore Database

  1. Clic en "Firestore Database"
  2. Clic en "Crear base de datos"
  3. "Modo de prueba"
  4. Elegir ubicación cercana

⚙️ Obtener configuración

  1. Clic en ⚙️ → "Configuración del proyecto"
  2. Bajar a "Tus apps" → Clic en "" (web)
  3. Nombre de app → "Registrar app"
  4. Copiar toda la configuración firebaseConfig
  5. Pegar en tu archivo config.js
🚀

Paso 4: ¡Probar tu App!

Vamos a verificar que todo funcione perfectamente

🎯 Abrir la aplicación

  1. Busca el archivo index.html
  2. Haz doble clic para abrirlo en tu navegador
  3. Deberías ver la pantalla de inicio de sesión

👤 Crear tu primera cuenta

  1. Clic en "Regístrate"
  2. Ingresa tu email y contraseña
  3. Clic en "Crear Cuenta"
  4. ¡Deberías entrar a la app!

🤖 Probar la IA

  1. Una vez dentro, clic en "Tips IA"
  2. Deberías recibir un consejo financiero
  3. ¡Si funciona, todo está perfecto!
